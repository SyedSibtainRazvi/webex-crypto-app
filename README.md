# WebX

## Description:
**WebX** is a realtime crypto tracker built in **React JS** that fetches data from **Coingeko API** using Axios. And the navigation is done using **React Router v6**.

### Technologies Used:
1) React JS.
2) Axios.
3) React Router v6.
4) Vanilla CSS.
